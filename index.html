The 3 functions

defineParameters() — return an array of parameter objects. Each becomes a live slider or input in the controls bar below the canvas.

{ name:"rate", type:"slider", min:0, max:1, default:0.3, step:0.01 } { name:"count", type:"number", min:1, max:1000, default:100 } { name:"wrap", type:"checkbox", default:true }

init(params) — called once on Play/Reset. Returns your initial entities array. params holds current slider values.

update(entities, params) — called ~60× per second. Returns the new entities array. Use spread: return entities.map(e => ({...e, x: e.x+e.vx}))

Entity properties

x, yCanvas position. Default canvas: 800×550.
shape'circle' | 'rect' | 'triangle'
radiusSize (rect uses radius × radius unless height set)
colorAny CSS color. 'transparent' skips drawing.
angleRotation in radians (triangles)
trailArray of {x,y} for motion trails
labelString drawn above entity
